    通用指令（參考）







    動作系統話機按鍵網頁介面
    進入設定# 或 0登入管理後台
    管理密碼預設 1234/0000admin/admin
    設定時間11 + [HHMM]Date/Time 頁面
    設定日期12 + [YYMMDD]Date 欄位

    ⚠️ 品牌差異提醒


  • 東訊：話機按 *# → 7 → 1（時間設定）

  • 通航：話機按 *99 進入設定模式

  • 眾通/萬國：多數需透過 # 或 * 進入

    3. 錄製語音設定

    流程圖


    ┌─────────────────┐
    │ 進入語音設定    │
    └────────┬────────┘
             ▼
    ┌─────────────────┐
    │ 選擇錄音功能    │
    └────────┬────────┘
             ▼
    ┌─────────────────┐     ┌─────────────────┐
    │ 選擇音軌編號    │────▶│ 開始錄音        │
    └────────┬────────┘     └────────┬────────┘
             │                        ▼
             │               ┌─────────────────┐
             │               │ 對著話機說話    │
             │               │ (約10-30秒)     │
             │               └────────┬────────┘
             │                        ▼
             │               ┌─────────────────┐
             │               │ 按 # 結束錄音   │
             │               └────────┬────────┘
             │                        ▼
             │               ┌─────────────────┐
             │               │ 播放確認         │
             └──────────────▶│ 如需重錄請重複  │
                             └─────────────────┘

    常用音軌編號










    音軌一般用途
    00 / 01上班時間總機語音
    02下班/非上班時間語音
    03午休時間語音
    04假日語音
    05忙碌中提示
    06無人接聽提示
    07語音留言提示

    錄音技巧


  • 環境：安靜、無回音

  • 語速：慢速、咬字清晰

  • 長度：建議 10-30 秒

  • 結束：說完後按 # 停止

    • > 📝 錄音腳本範例：
    > 

    > 「您好，感謝來電OO公司，現在是上班時間，請稍後，我們將為您轉接客服人員，
    >  若有需要留言，請按 9 留下訊息，謝謝。」
    >

    流程圖


    ┌─────────────────┐
    │ 進入時段設定    │
    └────────┬────────┘

             ▼
        ┌────┴────┐
        ▼         ▼         ┌─────────────┐
    ┌───────┐ ┌───────┐     │  設定時段    │
    │固定時段│ │自定義時│────▶ (每週循環)  │
    └───┬───┘ └───┬───┘     └─────────────┘
        ▼         ▼
    ┌───────┐ ┌───────┐
    │上班時段│ │下班時段│

        │         │
        └────┬────┘
             ▼
    ┌─────────────────┐
    │ 關聯語音檔      │
    └────────┬────────┘
             ▼
    ┌─────────────────┐
    │ 啟用/關閉時段   │
    └────────┬────────┘
             ▼
    ┌─────────────────┐
    │ 儲存並測試      │
    └─────────────────┘

    時段設定參考







    時段名稱預設時間對應語音
    上班時間09:00-18:00音軌 01
    下班時間18:00-09:00音軌 02
    午休時間12:00-13:00音軌 03
    假日週六/日音軌 04

    指令速查









    功能指令說明
    設定時段60 + [時段編號]進入時段設定
    設定開始時間61 + [HHMM]開始時間
    設定結束時間62 + [HHMM]結束時間
    設定假日63假日設定
    綁定語音64 + [音軌編號]關聯語音檔
    啟用時段65 + [0/1]0=關閉, 1=開啟

    週間設定範例


    星期一至星期五：
    ├── 上班：09:00-12:00 → 語音 01
    ├── 午休：12:00-13:00 → 語音 03
    ├── 上班：13:00-18:00 → 語音 01
    └── 下班：18:00-09:00 → 語音 02
    星期六、日：
    └── 假日：全天 → 語音 04

    設定完成後確認事項


  • [ ] 日期時間正確

  • [ ] 分機可互相呼叫

  • [ ] 語音檔可正常播放

  • [ ] 來電可轉接至分機/手機

    • 常見問題排除








    問題可能原因解決方式
    話機無訊號分機未註冊重新開機或重新註冊

    時段未切換時段設定錯誤檢查時間區段重疊
