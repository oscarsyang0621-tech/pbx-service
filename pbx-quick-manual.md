## 1. 日期時間設定

### 流程圖
```
┌─────────────────┐
│  進入設定模式   │
└────────┬────────┘
         ▼
┌─────────────────┐     ┌─────────────────┐
│ 系統話機按 *#   │────▶│ 輸入管理密碼    │
└────────┬────────┘     └────────┬────────┘
         │                        ▼
         │               ┌─────────────────┐
         │               │ 進入日期時間選單│
         │               └────────┬────────┘
         │                        ▼
         │               ┌─────────────────┐
         │               │ 設定年月日時分   │
         │               └────────┬────────┘
         │                        ▼
         │               ┌─────────────────┐
         │               │ 確認儲存         │
         └──────────────▶│ 離開設定         │
                         └─────────────────┘
```

### 通用指令（參考）

| 動作 | 系統話機按鍵 | 網頁介面 |
|------|-------------|----------|
| 進入設定 | `*#` 或 `*0` | 登入管理後台 |
| 管理密碼 | 預設 `1234`/`0000` | admin/admin |
| 設定時間 | `11` + [HHMM] | Date/Time 頁面 |
| 設定日期 | `12` + [YYMMDD] | Date 欄位 |

### ⚠️ 品牌差異提醒
- **東訊**：話機按 `*#` → `7` → `1`（時間設定）
- **通航**：話機按 `*99` 進入設定模式
- **眾通/萬國**：多數需透過 `#` 或 `*` 進入

> 📝 **請記錄**：你的設備型號：________ / 管理密碼：________

## 3. 錄製語音設定

### 流程圖
```
┌─────────────────┐
│ 進入語音設定    │
└────────┬────────┘
         ▼
┌─────────────────┐
│ 選擇錄音功能    │
└────────┬────────┘
         ▼
┌─────────────────┐     ┌─────────────────┐
│ 選擇音軌編號    │────▶│ 開始錄音        │
└────────┬────────┘     └────────┬────────┘
         │                        ▼
         │               ┌─────────────────┐
         │               │ 對著話機說話    │
         │               │ (約10-30秒)     │
         │               └────────┬────────┘
         │                        ▼
         │               ┌─────────────────┐
         │               │ 按 # 結束錄音   │
         │               └────────┬────────┘
         │                        ▼
         │               ┌─────────────────┐
         │               │ 播放確認         │
         └──────────────▶│ 如需重錄請重複  │
                         └─────────────────┘
```

### 常用音軌編號

| 音軌 | 一般用途 |
|------|---------|
| 00 / 01 | 上班時間總機語音 |
| 02 | 下班/非上班時間語音 |
| 03 | 午休時間語音 |
| 04 | 假日語音 |
| 05 | 忙碌中提示 |
| 06 | 無人接聽提示 |
| 07 | 語音留言提示 |

### 錄音技巧
- 環境：安靜、無回音
- 語速：慢速、咬字清晰
- 長度：建議 10-30 秒
- 結束：說完後按 `#` 停止

> 📝 **錄音腳本範例**：
> ```
> 「您好，感謝來電OO公司，現在是上班時間，請稍後，我們將為您轉接客服人員，
>  若有需要留言，請按 9 留下訊息，謝謝。」
> ```

## 4. 時段語音回覆設定（上班/下班/午休/假日）

### 流程圖
```
┌─────────────────┐
│ 進入時段設定    │
└────────┬────────┘
         ▼
┌─────────────────┐
│ 選擇時段類型    │
└────────┬────────┘
         ▼
    ┌────┴────┐
    ▼         ▼         ┌─────────────┐
┌───────┐ ┌───────┐     │  設定時段    │
│固定時段│ │自定義時│────▶ (每週循環)  │
└───┬───┘ └───┬───┘     └─────────────┘
    ▼         ▼
┌───────┐ ┌───────┐
│上班時段│ │下班時段│
│09:00-18:00│ │18:00-09:00│
└───┬───┘ └───┬───┘
    │         │
    └────┬────┘
         ▼
┌─────────────────┐
│ 關聯語音檔      │
└────────┬────────┘
         ▼
┌─────────────────┐
│ 啟用/關閉時段   │
└────────┬────────┘
         ▼
┌─────────────────┐
│ 儲存並測試      │
└─────────────────┘
```

### 時段設定參考

| 時段名稱 | 預設時間 | 對應語音 |
|---------|---------|---------|
| 上班時間 | 09:00-18:00 | 音軌 01 |
| 下班時間 | 18:00-09:00 | 音軌 02 |
| 午休時間 | 12:00-13:00 | 音軌 03 |
| 假日 | 週六/日 | 音軌 04 |

### 指令速查

| 功能 | 指令 | 說明 |
|------|------|------|
| 設定時段 | `60` + [時段編號] | 進入時段設定 |
| 設定開始時間 | `61` + [HHMM] | 開始時間 |
| 設定結束時間 | `62` + [HHMM] | 結束時間 |
| 設定假日 | `63` | 假日設定 |
| 綁定語音 | `64` + [音軌編號] | 關聯語音檔 |
| 啟用時段 | `65` + [0/1] | 0=關閉, 1=開啟 |

### 週間設定範例
```
星期一至星期五：
├── 上班：09:00-12:00 → 語音 01
├── 午休：12:00-13:00 → 語音 03
├── 上班：13:00-18:00 → 語音 01
└── 下班：18:00-09:00 → 語音 02

星期六、日：
└── 假日：全天 → 語音 04
```

## 📋 快速檢查清單

### 設定完成後確認事項
- [ ] 日期時間正確
- [ ] 分機可正常註冊
- [ ] 分機可互相呼叫
- [ ] 語音檔可正常播放
- [ ] 時段自動切換正常
- [ ] 來電可轉接至分機/手機

### 常見問題排除
| 問題 | 可能原因 | 解決方式 |
|------|---------|----------|
| 話機無訊號 | 分機未註冊 | 重新開機或重新註冊 |
| 語音無法播放 | 語音檔損壞 | 重新錄製 |
| 時段未切換 | 時段設定錯誤 | 檢查時間區段重疊 |
| 分機無法轉接 | 權限不足 | 檢查分機權限設定 |
